# Shib Owes You (SOU) Skill 🐾

This skill provides tools and information for navigating the Shib Owes You (SOU) ecosystem, which includes both Official SOU NFTs and the Community SOU Token.

## Core Knowledge Areas

1.  **Official SOU (Shib Owes You) NFTs**: On-chain recovery system for verified losses, tokenized as NFTs on Ethereum.
2.  **Community SOU Token**: A community-initiated liquid token, primarily on BSC, complementing the SOU NFT system and supporting educational initiatives.
3.  **SOU Domain Management**: Tools for querying SOU domain information and managing related assets.

## Usage Guide

Further details on specific functionalities, such as token balance queries and domain management, will be provided in dedicated reference files and scripts within this skill.

# Shib Owes You (SOU) Skill

This repository contains an OpenClaw skill for interacting with the Shib Owes You (SOU) ecosystem.

## Overview

The `shib-owes-you-skill` provides tools and information for developers and users to engage with SOU contracts and utilities across BSC, Ethereum, and Shibarium.

## Directory Structure

```
shib-owes-you/
├── SKILL.md          # Main skill definition and instructions
├── README.md         # This file
├── scripts/
│   └── sou_client.py # Python client for SOU ecosystem
└── tests/
    └── test_sou_client.py # Unit tests for the client
```

## Features

- Query $SOU and $SHIU token balances.
- Provides direct links for trading on major DEXs.
- Lists official contract addresses and utility tools.
- Information on minting `.sou` domains (ShibOwesYouID).

## Usage

To use this skill, load the `shib-owes-you` directory into an environment that supports the OpenClaw standard.

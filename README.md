# Shibarium Skills

A comprehensive collection of OpenClaw skills for the Shiba Inu ecosystem, focusing on Shibarium and related projects.

## Skills Included

- **Shibarium Skill**: Core network interaction tools for Shibarium Mainnet and Puppynet, including blockchain data queries, balance checks, and RPC endpoint interactions.
- **Shib Owes You (SOU) Skill**: Tools and information for the SOU ecosystem, covering Official SOU NFTs, Community SOU Token, and SOU domain management.

## Structure

Each skill is organized as follows:
- `SKILL.md`: Main skill definition and instructions.
- `scripts/`: Python scripts for contract interaction or utility functions.
- `tests/`: Unit tests for the scripts.
- `references/`: Detailed documentation, token addresses, network information, and other relevant data.
